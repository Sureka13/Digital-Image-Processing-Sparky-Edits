# Sparky Edits – Digital Image Processing Application

Sparky Edits is a feature-rich image editing application built using **Python, PyQt5, and OpenCV**.
It provides real-time image manipulation, pixel-level editing, drawing primitives, and color space transformations through an intuitive desktop GUI.

---

## Features
- Zoom, rotate, and crop images with live previews
- Draw shapes (lines, circles)
- Pixel-level color editing
- Add text with custom fonts and colors
- Color space conversions (Grayscale, HSV, LAB)
- Combine images horizontally or vertically
- Create blank canvases
- Thumbnail-based image management
- Supports PNG, JPG, BMP formats

---

## Technologies Used
- Python
- PyQt5
- OpenCV
- NumPy
